# Projeto de Análise de Código em Python

## Pré-requisitos

- **Python**: [Download e instalação do Python](https://www.python.org/downloads/)
- **Flask**: Instale o Flask usando o pip:
  
  ```
  pip install flask
  ```
- **SymPy**: Instale o SymPy usando o pip:
  
```
pip install sympy
```
  
## Executar projeto

```
python app.py
```
